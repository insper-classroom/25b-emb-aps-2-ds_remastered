Controle: Escudo para jogar Dark Souls (também funciona para outros jogos com mecânica de escudo)
Ideia: O IMU detecta o movimento do escudo e causa a ação de bloqueio, com um joystick + 4 botões no handle para complementar com o resto das mecânicas do jogo

- 4 botoes analogicos: ataque, esquiva, cura e pause
- imu: brandir o escudo (mover rapidamente) causa a ação de parry ou bloqueio no jogo
- joystick: movimentação básica (wasd)
